Para criar o controlador:
php artisan make: controller (nome do controller sempre no singular) JogoController

Do you want to run "make:controller" instead?  (yes/no) [no]:
> yes

A rota para o controller, não recebe function
fica:
Route::get('/nomeview',[|NomeController::class,'index']);

Ficou:
Route::get('/jogos',[JogoController::class,'index']);

index - nome de um método que será colocado no controller.

Para cada entidade que for criada, eu crio um controller para controlar.

O método index, será criado na controller. 

Na Classe JogoController
class JogoController extends Controller
{
    public function index(){
        
    }
}
